Cara Pemasangan

1 Backup file lama: sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
2 Edit file: sudo nano /etc/apt/sources.list
3 Ganti seluruh isi file dengan daftar yang Anda salin di atas.
4 Jalankan update: sudo apt update
